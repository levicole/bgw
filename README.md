# BGW
Just a web based git repo project

## Installation

This is still very new. But if you want to give it a try you give this a shot:

```
    git clone git@github.com:levicole/bgw.git
    cd bgw
    rake install
```

## Usage

```
    cd my/repository
    bgw .
```
